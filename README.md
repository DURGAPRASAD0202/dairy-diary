# 🥛 Dairy Diary - Milk Delivery & Payment Tracker

A modern, mobile-first web application designed for dairy distributors and milk delivery services to manage their daily operations with ease. 

## 🚀 Features

- **🏠 Today's Deliveries**: Quickly mark customers as "Delivered" or "Not Delivered" with a single tap.
- **👥 Customer Management**: Maintain a detailed database of customers, including their daily milk quantity, price per liter, and language preferences.
- **💰 Payment Tracking**: Track outstanding balances, record payments (Cash/UPI), and view payment history.
- **📊 Reports & Insights**: Generate monthly summaries, export delivery calendars as PDFs, and track total revenue vs. pending amounts.
- **🌐 Multi-language Support**: Supports English, Telugu, and Hindi for better accessibility.
- **📱 PWA Ready**: Installable on mobile devices for an app-like experience on the go.

## 🛠️ Tech Stack

- **Frontend**: Next.js (React), TypeScript, Vanilla CSS
- **Backend**: Firebase (Firestore, Authentication, Cloud Functions)
- **Utilities**: jsPDF (for receipts), Date-fns (for scheduling)

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/DURGAPRASAD0202/dairy-diary.git
   cd dairy-diary
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Configure Environment**:
   Create a `.env.local` file and add your Firebase configuration.

4. **Run the development server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## 🏃‍♂️ Quick Start (Windows)
Double-click the `run_app.bat` file in the root directory to automatically start the development server and open the dashboard in your browser.
